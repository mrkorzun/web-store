# 🛒 Full-Stack Online Store

Индивидуальный проект по разработке интернет-магазина на основе **DummyJSON
API**. <br> В проекте реализована логика работы с асинхронными запросами,
фильтрацией, поиском и управлением состоянием корзины.

## 🚀 Основной функционал

- **Каталог товаров:** Загрузка данных с API с поддержкой пагинации (12 товаров
  на страницу) и кнопкой "Load More".
- **Поиск и фильтрация:** Поиск по названию и фильтрация товаров по категориям с
  динамическим выделением активных элементов.
- **Wishlist:** Добавление товаров в список желаний с сохранением состояния в
  `localStorage`.
- **Корзина (Cart):** Добавление товаров, удаление, автоматический расчет общего
  количества (Items) и финальной стоимости (Total).
- **Детальная информация:** Модальное окно с подробным описанием товара,
  проверкой наличия в корзине и сменой текста кнопок (Add/Remove).
- **UI/UX:** Светлая и темная темы (с сохранением в `localStorage`), индикатор
  загрузки, кнопка ScrollUp и уведомления через библиотеку `iziToast`.

## 🔗 API Endpoints

В проекте используются следующие эндпоинты
[DummyJSON](https://dummyjson.com/docs/products):

- `GET /products` — получение списка всех товаров с пагинацией.
- `GET /products/{id}` — получение детальной информации о конкретном товаре.
- `GET /products/search?q={query}` — поиск товаров по ключевому слову.
- `GET /products/category-list` — получение списка всех доступных категорий.
- `GET /products/category/{name}` — фильтрация товаров по категории.

## 🛠 Технологический стек

- **JavaScript:** ES6+ модульная архитектура (import/export).
- **Верстка:** HTML5 (семантика), CSS3 (Flexbox/Grid).
- **API:** [DummyJSON](https://dummyjson.com/).
- **Библиотеки:** [iziToast](https://izitoast.marcelo-miranda.dev/) для
  уведомлений.

## 📁 Модульная структура JS

Код разделен на независимые модули для легкой поддержки и масштабируемости:

- `products-api.js` — взаимодействие с бекендом (fetch запросы).
- `render-functions.js` — создание HTML-разметки элементов.
- `storage.js` — функции для работы с `localStorage`.
- `handlers.js` — логика обработки событий (хендлеры).
- `modal.js` — управление состоянием модального окна.
- `refs.js` — централизованное хранение ссылок на DOM-элементы.

## 💻 Запуск проекта

1. Клонируйте репозиторий на свой ПК.
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите проект в режиме разработки:
   ```bash
   npm run dev
   ```
